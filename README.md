## Music Controller App (from Tech with Tim's Django & React Tutorial)

### Run this app:
 - run Django server:
    - activate the pipenv venv
    - `cd music_controller`
    - `python manage.py runserver`
 - run React frontend:
    - `cd music_controller/frontend`
    - `npm run dev`
